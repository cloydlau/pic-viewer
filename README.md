### Feature

- √ viewerjs的vue版本 增加了预览前的外部展示 使其开箱即用
- √ 支持自适应瀑布流展示、嵌套在表格内展示
- √ 灵活的数据类型：支持字符串/字符串数组/对象数组
- √ 友好的hover交互
- √ 支持全局安装/单独引入


### Installation
![NPM](https://nodei.co/npm/pic-viewer.png)
```
yarn add pic-viewer / npm i pic-viewer
Dependencies：vue

全局引入：
import PicViewer from 'pic-viewer'
Vue.use(PicViewer)

局部引入：
import { PicViewer } from 'pic-viewer'
components: { PicViewer }
```
